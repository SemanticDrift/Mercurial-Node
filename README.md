## What This Does

Derives Mercury's orbital architecture from first principles using only raw observational data and exact rational arithmetic. No gravitational constants, mass parameters, or curve-fitting adjustments are introduced. The terrestrial baseline provides the reference frame.

The analysis establishes Mercury as the high-frequency terminal node of a standing-wave lattice with system constant k=5 and admissibility lock at n=20. The semi-major axis (0.3871 AU), 3:2 spin-orbit resonance, and axial tilt (2.11°) emerge as geometric consequences of wave-phase limits near the solar primary. Four classical anomalies are resolved from the lattice: perihelion precession (43.02 arcsec/century), high-density iron core (60-70%), 3:2 resonance, and crustal contraction (6.05 km). All derivations use the Origami Principle (22/7) in place of π.

This annex completes the architectural map of the planetary system, resolving the innermost boundary node of the solar field.
